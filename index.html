Reproducible Analysis (Python)


import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# Load simplified revenue series
rev = pd.Series([2.9, 3.0, 3.1, 3.2], index=["Q3'24","Q4'24","Q1'25","Q2'25"]).pct_change()

# Rolling growth (QoQ)
roll = (1 + rev).rolling(2).apply(np.prod) - 1
print("2Q rolling growth:", f"{roll.iloc[-1]:.2%}")

Financial Math

Net Present Value (NPV):

\[ \text{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1+r)^t} - C_0 \]

Capital Asset Pricing Model (CAPM):

\[ \mathbb{E}[R_i] = R_f + \beta_i\,(\mathbb{E}[R_m]-R_f) \]

Explain discount rate rationale and link to WACC. CAPM sets required return; tie back to hurdle rates.
